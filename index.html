A little about me

I’ve made my home base in Hilo, Hawaii for the past few years. I live in a small apartment on the bay with my wonderful fiancee Joe, where we live and work with the salty ocean air. After becoming part of a team of like-minded buisiness owners, I discovered that we all had a need for one thing and we needed that thing soon. That one thing was a website. A web presence is crucial in today’s marketing arena and so I jumped into coding with both feet!!

Since I built my first Wordpress website using tutorials on Youtube, I have graduated to taking more structured courses at Skillcrush and I am learning so quickly it seems impossible!! My first desire is to be able to build a quick, affordable, and easily updated website for my fellow entrepreneurs to be able to install & update easily.
